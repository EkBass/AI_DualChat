## **Getting Started**

### **Prerequisites**
- Node.js (v16 or newer)
- An OpenAI API key ([Get yours here](https://platform.openai.com/signup/))

---

### **Installation**
1. Clone the repository:
   ```bash
   git clone https://github.com/EkBass/AI_DualChat.git
   cd AI_DualChat
   ```
2. Run as:
   ```bash
   node AI_DualChat.js
   ```
